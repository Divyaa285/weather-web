# Weather Forecast Web App (HTML, CSS & JavaScript)

A frontend-based Weather Forecast Web Application built using **HTML, CSS, and JavaScript**.  
The application allows users to search for any city, view current weather details, check a 5-day forecast, and manage favorite cities with smooth dashboard navigation.

## Key Features
--City search functionality
--Current weather display
--5-day weather forecast
--Navigation between screens
--Error handling and responsive UI
--Favorite cities using localStorage


## What It Does
Shows today's weather (temperature & wind)
Shows weather for next 5 days
Let you save favorite cities
Works on phone and computer
